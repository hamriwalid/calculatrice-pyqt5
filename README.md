# PyCalc - Calculatrice Graphique Moderne en Python

Une application de calculatrice de bureau esthétique et réactive, développée en **Python** avec la bibliothèque **PyQt5**.

## Fonctionnalités
- **Design Moderne :** Interface épurée en mode sombre (Dark Mode).
- **Interactivité :** Effets de style dynamiques au survol (`hover`) et au clic des boutons grâce à **QSS** (Qt Style Sheets).
- **Opérations de Base :** Addition (`+`), Soustraction (`-`), Multiplication (`*`), et Division (`/`).
- **Gestion des Erreurs :** Empêche le plantage de l'application en cas de mauvaise syntaxe ou de division par zéro (affiche "Erreur").


## Technologies Utilisées
- **Langage :** Python 3.x
- **Interface Graphique :** PyQt5
- **Style :** QSS (Qt Style Sheets / CSS pour Qt)
